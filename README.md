# KVHALO-Inference
KVHALO is a autoregressive AI model that will intercept KV caches to upscale them dynamically. This is possible through the inference scripts and examples provided. 

## Minimum Requirements for Inference:
- A GPU with atleast CUDA drivers - size determines on the model you are using. 
- Huggingface AI model ([https://huggingface.co/richyvd/KVHalo])
- Requirements in requirements.txt
- The Original AI Model that you trained it on
## Installation
```
pip install -r requirements.txt
```
